# Calculatrice Scientifique avec RMI

##  Description
Calculatrice scientifique complète utilisant **Java RMI** pour la communication client-serveur.
L'interface graphique est réalisée avec **JavaFX**.

##  Fonctionnalités

| Onglet | Description |
|--------|-------------|
| **Calcul** | Calculatrice scientifique (sin, cos, tan, ln, log, √, xʸ, n!, mémoire) |
| **Courbe** | Tracé de fonctions mathématiques |
| **Polynôme** | Résolution de polynômes degré 1 à 5 |
| **Comparer** | Comparaison de 3 courbes avec intersections |
| **Stats** | Statistiques (moyenne, médiane, écart-type, quartiles) |
| **Unités** | Conversion d'unités (longueur, masse, données, etc.) |
| **Matrice** | Opérations sur matrices 2x2, 3x3, 4x4 |

##  Architecture
- **Serveur RMI** : exécute les calculs
- **Client** : interface graphique

##  Technologies
- Java 21+
- JavaFX
- RMI

## ▶️ Lancement
1. Lancer `ServeurGUI.java`
2. Lancer `ClientGUI.java`
3. Cliquer sur "CONNECTER"
